# search terms



```
#filters results only from the given domain
site:{site}

#filters results of sites that contain the term in the URL
inurl:{term}

#filters similar sites to the given
related:{site}

#searches for the term within sites or documents
intext:{term}

#displays sites that have a link to the given site
link:{site}

#filters the documents that contain the given word in the title
intitle:{term}
```
